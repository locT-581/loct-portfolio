A high-performance, aesthetically driven portfolio website architected for modern web standards. Built with **Next.js 16**, **Tailwind CSS 4**, and **Framer Motion**, this project showcases a seamless fusion of engineering excellence and creative design.

---

## ✨ Key Features

- 🌐 **Dual-Language i18n**: Fully translated in English & Vietnamese using `next-intl`.
- 🌊 **Smooth Scrolling**: Integrated with **Lenis Scroll** for a premium, fluid navigation experience.
- 🎨 **Luxury Animations**:
  - **Scroll Reveal Text**: Character-by-character reveals synchronized with scroll progress.
  - **Premium Counters**: Animated statistics and counters that trigger on view.
  - **Staggered Entrance**: Coordinated micro-interactions for grid items and sections.
  - **Interactive Services**: Fluid background and content shifts on hover.
- 🌓 **Dynamic Theme**: Full Light/Dark mode support via `next-themes`.
- ⚡ **Optimized Performance**: Leverages Next.js App Router, Server Components, and optimized font loading.
- 📱 **Fully Responsive**: Pixel-perfect design adapted for mobile, tablet, and desktop screens.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v20 or higher)
- [npm](https://www.npmjs.com/) or [pnpm](https://pnpm.io/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/locT-581/loct-portfolio.git
   cd loct-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   npm start
   ```
